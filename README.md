# Md Rtl Viewer

A clean, minimal Markdown editor with live preview and full RTL support.

---

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/bakrialloush/md-rtl.git
   cd md-rtl
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the editor:
   ```bash
   npm run dev
   ```

4. Open your browser and go to `http://localhost:5173`

---

## User Guide

### Writing Markdown

Type your Markdown in the left pane. The right pane updates instantly as you write.

### Switching Text Direction

Click the **RTL / LTR** button in the preview toolbar to toggle between right-to-left and left-to-right text direction.

### Changing the Theme

Click any of the colored dots in the preview toolbar to switch the accent color theme.

### Sync Scroll

Enable **sync scroll** in the top bar to keep the editor and preview scrolling in sync as you write.

### Copying Content

- **Markdown pane** — click **Copy** to copy the raw Markdown text.
- **Preview pane** — click **Copy** to copy the rendered content as rich text, ready to paste into any document editor.

### Exporting

Both export buttons are in the preview toolbar:

- **Export HTML** — downloads a self-contained `.html` file with all styles embedded. Open it in any browser.
- **Export PDF** — opens the browser's print dialog. Choose **Save as PDF** to save a PDF file.
